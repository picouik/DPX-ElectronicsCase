# DPX Electronics Case

Version: 1.0.0  
Compatibility: SPT 4.0.13 build 40087

DPX Electronics Case adds a dedicated electronics storage case for SPT. It uses a custom graphite/PCB texture, a validated stash/trader icon, and a strict electronics whitelist.

## Features

- Sold by Skier LL1
- Price: 250000 roubles
- External size: 3x3
- Internal storage: 10x10
- Server-side mod only
- No BepInEx plugin
- No SPT_Data changes

## Accepted Items

The case accepts electronics and advanced technology components, including:

- CPU
- CPU fan
- RAM
- SSD
- HDD
- Portable Powerbank
- Printed Circuit Board
- Military Cable
- Military Circuit Board
- VPX
- Virtex
- RFID
- COFDM
- Microcontroller Board

## Installation

Download the release ZIP and extract it into your SPT root folder.

The ZIP contains:

```text
SPT/user/mods/DPX-ElectronicsCase/
```

After extraction, start the SPT server and check Skier LL1.

## Uninstallation

Delete:

```text
SPT/user/mods/DPX-ElectronicsCase
```

## Notes

This mod does not modify `SPT_Data` or vanilla game files.
